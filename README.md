# Lingo-Matrix: Gemini-Powered Oral English Intelligence

## 🚀 项目概述
Lingo-Matrix 是一个基于 **Gemini 3.1 Pro** 的原生多模态英语口语系统。
它通过 **Multi-Agent 协作** 和 **百万级 Token 长上下文** 技术，解决了 AI 口语教练“没记性”和“逻辑浅”的痛点。

## ✨ 核心特性
- **Long-Context Memory:** 支持挂载百万量级历史语料，实现跨周期的个性化教学反馈。
- **CoT Reasoning:** 后台 Coach Agent 采用思维链推理，深度剖析用户语言习得曲线。
- **Multimodal Integration:** 实时监测语音语调，捕捉情感波动。

## 🛠️ 技术栈
- **Model:** Gemini 3.1 Pro (Experimental) / Gemini 1.5 Pro
- **Orchestration:** LangGraph / OpenClaw
- **Backend:** FastAPI + Python 3.11
